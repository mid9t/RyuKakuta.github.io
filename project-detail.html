Epic Ranker, Part 1: Using Evidential Uncertainty to Measure When BERT Should Hesitate

Introduction

Modern language models often produce confident predictions even when inputs differ from the data they were trained on. In ranking and decision systems, that behavior is risky: the model may look certain precisely when it should defer, request more computation, or abstain.

Epic Ranker is an uncertainty-aware ranking architecture designed to allocate computation dynamically. Rather than using a fixed amount of inference for every example, it estimates uncertainty and uses that signal to support later-stage routing, early exit, or escalation.

Epic Ranker is a project exploring dynamic computational allocation through evidential uncertainty learning. The central idea is simple: instead of treating every input equally, the system should spend more computation on ambiguous cases and less on easy ones.

Phase 1 focuses on the foundation for that idea. I attach a Dirichlet-based evidential head to a pretrained BERT encoder and evaluate whether it produces meaningful uncertainty signals under out-of-distribution and corrupted inputs.

Methodology

We employed Granger Causality to establish effective connectivity between EEG channels, creating connectivity matrices that served as input to Convolutional Neural Networks. The CNN architecture was designed to capture spatial patterns in the connectivity data while being computationally efficient.

Key Challenges

  • Preventing data leakage in time-series cross-validation
  • Ensuring model interpretability for clinical acceptance
  • Handling the high dimensionality of connectivity matrices
  • Validating performance on unseen, real-world data

Results

The model achieved promising classification accuracy while maintaining interpretability. Feature importance analysis revealed key connectivity patterns that aligned with neuroscientific understanding of depression-related brain activity changes.

Impact

This work demonstrates the potential of machine learning in mental health diagnostics, emphasizing the importance of rigorous evaluation practices in high-stakes medical applications.
